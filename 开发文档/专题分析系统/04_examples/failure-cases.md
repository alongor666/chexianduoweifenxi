## 失败用例 1:看似专业的赔付率归因分析

**用例内容**

"2024年1-11月,分公司综合赔付率同比上升2.8个百分点,主要原因是新能源车占比提升。新能源车出险频度高于传统燃油车,且零整比较高导致案均赔款上升。建议加强新能源车承保筛选。"

**v1.0 拦截分析**

| Gate | 问题 |
|-----|------|
| Gate 2→3 | 未标注赔付率口径与对比基准 |
| Gate 3→4 | 未量化新能源车贡献度 |
| Gate 4→5 | 仅有单一假设 |
| Gate 5→6 | 推断未给证据 |

## 失败用例 2:看似合理的费用管控建议

**用例内容**

"综合费用率超预算 1.6pct,主要是代理手续费高。建议下调手续费上限,加大直销投入。"

**v1.0 拦截分析**

| Gate | 问题 |
|-----|------|
| Gate 3→4 | 手续费贡献度未量化 |
| Gate 4→5 | 未说明高费率机制 |
| Gate 7→完成 | 对策与结论无映射、前提未校验 |
