# 压力测试结果(Pressure Tests)

- 版本:v0.1
- Phase:Phase 2
- 状态:Draft
- 适用范围:车险经营专题分析(配套《专题分析统一母模板 v1.0》)
- 日期:2025-12-13

---

## 测试专题类型 A:指标恶化但规模增长的矛盾型专题

**典型场景**
保费同比增长 15%,但综合赔付率同比恶化 3 个百分点。

**逻辑跳高发点**
直接从"赔付率恶化"跳到"业务质量下降"或"理赔管控不力"。

**v1.0 模拟运行与拦截点**

| 节点 | 模拟状态 | Gate 检查 |
|------|----------|-----------|
| 节点 1 | 边界可锁定 | Gate 1→2 通过 |
| 节点 2 | 现象陈述"赔付率恶化 3pct" | Gate 2→3 检查口径三要素 |
| 拦截点 | 若未标注"满期赔付率/已决赔付率""滚动12月/累计""同比基准",Gate 2→3 不通过 | |
| 节点 3 | 需拆解:险种结构、车型结构、出险频度、案均赔款 | Gate 3→4 检查贡献度量化 |
| 拦截点 | 若直接说"主要是新能源车拉高"但无贡献度量化,Gate 3→4 不通过 | |
| 节点 4 | 多假设:险种结构变化 / 车型结构变化 / 出险频度上升 / 案均上升 / 估损偏差 | Gate 4→5 检查 |
| 拦截点 | 若仅列"理赔管控不力"单一假设,Gate 4→5 不通过 | |

**如果没有 v1.0 会发生的逻辑跳**
- 跳过结构拆解,直接归因于"业务质量"
- 跳过假设竞争,直接锁定单一原因
- 对策指向"加强理赔管控",但真实原因可能是险种结构迁移

## 测试专题类型 B:多机构表现分化但无明显单因子的专题

**典型场景**
5 家机构综合成本率从 92% 到 108% 分化,但保费规模、渠道结构、车型结构均无单一解释变量。

**逻辑跳高发点**
选择性归因:只分析表现最差的机构,用其特征"反推"为原因。

**v1.0 模拟运行与拦截点**

| 节点 | 模拟状态 | Gate 检查 |
|------|----------|-----------|
| 节点 2 | 现象陈述需覆盖"分化"本身 | Gate 2→3 检查 |
| 拦截点 | 若现象仅描述最差机构,Gate 2→3 不通过 | |
| 节点 3 | 拆解需覆盖全部机构并量化贡献 | Gate 3→4 |
| 拦截点 | 若只拆解 1–2 家机构,Gate 3→4 不通过 | |
| 节点 4 | 假设需解释"为何分化" | Gate 4→5 |
| 拦截点 | 若假设仅针对单一机构,Gate 4→5 不通过 | |

**如果没有 v1.0 会发生的逻辑跳**
- 把"分化问题"偷换为"最差机构问题"
- 用单点反推整体原因
- 对策无法覆盖系统性问题

## 测试专题类型 C:结论高度依赖口径选择的专题

**典型场景**
签单口径保费增长 12%,满期口径保费下降 3%。

**逻辑跳高发点**
选择对自己有利的口径,不说明口径差异原因。

**v1.0 模拟运行与拦截点**

| 节点 | 模拟状态 | Gate 检查 |
|------|----------|-----------|
| 节点 1 | 边界需锁定口径 | Gate 1→2 |
| 拦截点 | 边界未锁定口径,Gate 1→2 不通过 | |
| 节点 2 | 现象需标注口径 | Gate 2→3 护栏 A |
| 拦截点 | 未标注签单/满期,Gate 2→3 不通过 | |
| 节点 6 | 结论需声明口径适用性 | Gate 6→7 |
| 拦截点 | 未声明口径边界,Gate 6→7 不通过 | |

**如果没有 v1.0 会发生的逻辑跳**
- 结论随口径切换而变化
- 第三方无法复核
- 决策被误导
