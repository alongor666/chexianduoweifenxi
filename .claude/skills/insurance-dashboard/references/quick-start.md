## 📊 理解输出结果

### 核心指标卡片

- **总保费规模**: 签单保费总和(亿元)
- **保单件数**: 承保保单总数(万件)
- **赔付率**: 已发生赔款 / 满期保费
- **费用率**: 费用金额 / 签单保费
- **综合成本率**: 赔付率 + 费用率 (< 100% 为盈利)
- **边际贡献**: 保费 - 赔款 - 费用 (万元)

### 健康度判断标准

| 指标 | 优秀 | 良好 | 合格 | 需改进 |
|------|------|------|------|--------|
| 赔付率 | < 60% | 60-70% | 70-85% | > 85% |
| 费用率 | < 25% | 25-30% | 30-35% | > 35% |
| 综合成本率 | < 95% | 95-100% | 100-105% | > 105% |

## 🔄 自动化工作流

### 每周自动生成报告

创建一个自动化脚本 `weekly_report.sh`:

```bash
#!/bin/bash
# 获取最新数据
WEEK=$(date +%V)
DATA_FILE="2025保单第${WEEK}周变动成本明细表.csv"

# 生成报告
python scripts/generate_dashboard.py \
    "$DATA_FILE" \
    comprehensive \
    > "outputs/week_${WEEK}_report.html"

echo "第 ${WEEK} 周报告已生成"
```

## 🎯 最佳实践

1. **每周定期分析** - 建立固定的数据更新和分析节奏
2. **关注异常值** - 重点关注预警标记的业务

4. **深入钻取** - 从宏观到微观,逐层分析问题
5. **数据驱动决策** - 基于分析结果制定业务策略
