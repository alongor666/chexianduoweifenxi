# 华安保险车险业务领域知识库

> 针对华安保险四川分公司车险业务的领域知识参考

## 一、核心术语表

### 1.1 保费相关

| 术语           | 英文                     | 定义                                         | 示例                       |
| -------------- | ------------------------ | -------------------------------------------- | -------------------------- |
| 签单保费       | Written Premium          | 保险公司在一定时期内承保的全部保单的保费总额 | 本周签单保费150万元        |
| 年累计保费     | YTD Premium              | 年初至今的保费累计值                         | 年累计保费5000万元         |
| 保费计划达成率 | Premium Achievement Rate | 实际保费 / 计划保费 × 100%                   | 达成率95%表示完成计划的95% |

### 1.2 成本与赔付

| 术语       | 英文                | 定义                                | 阈值标准           |
| ---------- | ------------------- | ----------------------------------- | ------------------ |
| 变动成本率 | Variable Cost Ratio | (满期赔付 + 费用) / 签单保费 × 100% | <90%达标，>95%预警 |
| 满期赔付率 | Ultimate Loss Ratio | 满期赔款 / 满期保费 × 100%          | <70%良好，>75%偏高 |
| 费用率     | Expense Ratio       | 费用支出 / 签单保费 × 100%          | <18%达标，>20%偏高 |
| 已报告赔款 | Reported Claims     | 已经报案的赔款金额                  | 用于损失暴露分析   |

### 1.3 风险指标

| 术语     | 英文                 | 定义                       | 行业平均       |
| -------- | -------------------- | -------------------------- | -------------- |
| 出险率   | Claim Frequency      | 出险件数 / 承保件数 × 100% | 20-25%         |
| 案均赔款 | Average Claim Amount | 总赔款 / 出险件数          | 家用车约6000元 |

**分析用途**:

- 识别高风险客户类别
- 定价策略优化
- 承保政策调整

## 三、麦肯锡风格报告规范

### 3.1 问题导向标题

**基本原则**:

- 标题即结论：一句话说清楚问题
- 突出行动点：指出需要关注的机构/类别
- 量化表达：用具体数字增强说服力

**标题模板**:

```
格式：{整体评价}，{问题机构1}、{问题机构2}机构{具体问题}

✅ 好的示例：
- "本周成本控制良好(85.3%)，高新机构保费进度落后"
- "高新机构变动成本率偏高(98.2%)，需加强成本管控"
- "家用车赔付率上升至76.5%，高于行业均值70%"

❌ 不好的示例：
- "经营分析"（无信息量）
- "各机构情况"（过于笼统）
- "第49周数据"（未提炼结论）
```

### 3.2 视觉规范

**配色标准**:

- 主色：深红 `#a02724` （华安品牌色）
- 辅助色：灰度（`#404040`, `#808080`, `#c0c0c0`）
- 状态色：
  - 绿色 `#00b050` - 良好
  - 黄色 `#ffc000` - 预警
  - 红色 `#c00000` - 严重

**字体规范**:

- 标题：微软雅黑 Bold 24pt
- 核心数字：Arial Bold 48pt
- 正文：微软雅黑 14pt
- 图表标注：微软雅黑 12pt

**布局规范**:

- 页面尺寸：16:9 宽屏（13.333 × 7.5英寸）
- 页边距：0.8英寸
- 顶部装饰：红色细线（0.015英寸高）
- 留白原则：页面留白 > 40%

### 3.3 图表类型与使用场景

#### 四象限图（Quadrant Chart）

**使用场景**:

- 双指标交叉分析
- 识别问题象限
- 机构/类别对比

**标准设置**:

```python
X轴：保费达成率（基准线100%）
Y轴：变动成本率（基准线90%）
气泡大小：签单保费规模

四个象限：
- 右下（绿）：双达成（保费高、成本低）
- 右上（黄）：保费达标但成本高
- 左下（黄）：成本达标但保费低
- 左上（红）：双落后（保费低、成本高）
```

**视觉要点**:

- 基准线使用灰色虚线
- 问题象限的点标红
- 直接在点旁标注机构名
- 无需图例（信息已在图上）

#### 气泡图（Bubble Chart）

**使用场景**:

- 三维数据展示
- 突出权重差异
- 损失暴露分析

**标准设置**:

```python
X轴：满期赔付率
Y轴：当年已报告赔款占比
气泡大小：当周已报告赔款占比

颜色：
- 问题机构：红色
- 正常机构：灰色
```

## 四、数据处理规范

### 4.1 数据质量检查

**必需字段检查**:

```python
required_fields = [
    '机构', '客户类别', '签单保费',
    '满期赔付率', '费用率', '变动成本率',
    '已报告赔款', '出险率', '案均赔款'
]
```

**数据类型验证**:

- 机构、客户类别：字符串
- 保费、赔款：数值（>0）
- 率值指标：百分比（0-100或0-1）

**异常值处理**:

```python
# 成本率 > 100% 警告但继续
if cost_ratio > 100:
    logger.warning(f"{org}成本率超过100%，数据可能有误")

# 保费 < 0 直接报错
if premium < 0:
    raise ValueError(f"{org}保费为负值，数据错误")
```

### 4.3 支持的数据格式

#### Excel (.xlsx, .xls)

**优点**: 最常用，支持格式化
**读取方式**: `pd.read_excel()`
**注意事项**:

- 跳过标题行（通常前2-3行）
- 处理合并单元格
- 识别多sheet场景

#### CSV (.csv)

**优点**: 通用性强
**读取方式**: `pd.read_csv(encoding='utf-8')`
**注意事项**:

- 编码问题（GBK vs UTF-8）
- 分隔符（逗号 vs 制表符）
- 引号转义

#### JSON (.json)

**优点**: 结构化，API友好
**读取方式**: `pd.read_json()`
**格式要求**: 数组或对象数组

#### DuckDB (.db, .duckdb)

**优点**: 高性能，支持大数据量
**读取方式**: `duckdb.connect().execute().df()`
**表名约定**: `insurance_data` 或第一个表

## 六、常见问题处理

### Q1: 数据中有特殊字符或乱码

**原因**: 编码问题（GBK vs UTF-8）

**解决方案**:

```python
# 尝试多种编码
encodings = ['utf-8', 'gbk', 'gb2312']
for enc in encodings:
    try:
        df = pd.read_csv(file_path, encoding=enc)
        break
    except UnicodeDecodeError:
        continue
```

### Q2: 机构名称不统一

**原因**: 手工录入或系统差异

**解决方案**:

```python
# 模糊匹配合并
org_mapping = {
    '高新': '高新机构',
    '高新支公司': '高新机构',
    'GX': '高新机构'
}
df['机构'] = df['机构'].replace(org_mapping)
```

### Q3: 缺少保费计划数据

**原因**: plans.json文件不存在

**处理策略**:

1. 提示用户："检测到无保费计划配置，保费进度分析页将跳过"
2. 继续生成其他页面
3. 在报告中标注："保费达成率数据缺失"

### Q4: 图表生成失败

**常见原因**:

- matplotlib中文字体缺失
- 数据为空或全0
- 坐标轴范围异常

**解决方案**:

```python
# 配置中文字体
plt.rcParams['font.sans-serif'] = ['Microsoft YaHei']
plt.rcParams['axes.unicode_minus'] = False

# 数据检查
if df.empty or df['签单保费'].sum() == 0:
    logger.warning("数据为空，跳过图表生成")
    return None
```
