## 核心功能
1. [功能点1]
2. [功能点2]
3. [功能点3]

## 技术实现

### 数据结构
[相关的数据类型、接口定义]

### 核心组件

- Hook: `src/hooks/use-xxx.ts`
- 服务: `src/services/XXXService.ts`（如适用）

### 计算逻辑（如涉及KPI）
[引用 core_calculations.md 中的公式]

## 使用示例
\```typescript
// 代码示例
\```

## 测试要点
- [ ] 基础功能验证
- [ ] 边界条件测试
- [ ] 性能测试（如适用）

## 相关文档

- KPI计算: `开发文档/03_technical_design/core_calculations.md`（如适用）

- 2025-XX-XX: 初始版本

## 更新文档时的注意事项

1. **使用中文编写所有文档**
2. **保持文档结构清晰**
3. **代码示例要准确且可运行**
4. **优先使用 `Edit` 工具修改现有文档**
5. **不要创建新文档，除非功能确实是全新的**
6. **归档过时文档到 `archive/` 目录**
7. **版本标注**：
   - 技术文档需标注版本号和状态
   - 状态标识: ✅ 现行标准 / 📝 草稿 / 🗄️ 已归档
   - 更新历史必须记录
