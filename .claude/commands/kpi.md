# KPI 计算命令 (/kpi)

基于周六截止的年度累计 CSV 数据，快速计算 16 个核心 KPI 指标。

## 用法

```
/kpi [文件路径] [选项]
```

## 参数

- `文件路径` - CSV 数据文件路径（可选，默认使用最近上传的文件）
- `--mode=cumulative` - 当周值模式（默认）
- `--mode=incremental` - 周增量模式（需要两周数据）
- `--week=N` - 指定周次

## 快速启动

```
/kpi                           # 使用最近上传的数据
/kpi data/第45周.csv           # 指定文件
/kpi --mode=incremental        # 周增量模式
```

## 计算的 16 个 KPI

### 核心比率 (第一行)
1. 满期边际贡献率 (%)
2. 保费时间进度达成率 (%)
3. 满期赔付率 (%)
4. 费用率 (%)

### 核心金额 (第二行)
5. 满期边际贡献额 (万元)
6. 签单保费 (万元)
7. 已报告赔款 (万元)
8. 费用额 (万元)

### 结构效率 (第三行)
9. 变动成本率 (%)
10. 满期率 (%)
11. 满期出险率 (%)
12. 保单件数 (件)

### 单均质量 (第四行)
13. 赔案件数 (件)
14. 单均保费 (元)
15. 案均赔款 (元)
16. 单均费用 (元)

## 关键规则

1. **数据是累计值**: CSV 中的数值是年初至今累计，不是单周发生额
2. **比率用累计计算**: 满期赔付率等比率必须基于累计数据
3. **周六截止**: 每周数据截止到周六
4. **50周工作制**: 周计划 = 年度目标 ÷ 50

## 关联技能

此命令调用 `kpi-calculator` 技能执行实际计算。
