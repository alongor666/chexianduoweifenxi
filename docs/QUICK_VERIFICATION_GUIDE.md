# 快速确认指南

本速查表提供命令参考与常见查询示例，帮助你在几分钟内确认 AgentDB 的学习是否生效。

## 常用命令
- **写入记忆**：`agentdb remember "内容"`
- **检索记忆**：`agentdb recall --query "问题"`
- **技能调用**：`agentdb skill <skill-name> --args "参数"`
- **因果追溯**：`agentdb causal --query "为什么/如何"`
- **查看技能库**：`agentdb skills list`
- **清理记忆**：`agentdb memory purge --before <日期>`

## 快速确认套路
1. **单条写读**：写入一句话后立即检索，预期完全复述且带来源引用。
2. **多轮上下文**：连续提问同一主题，确认第二轮回答会引用第一轮上下文而非重新生成。
3. **技能命中**：调用一个带明确输入输出的技能（例如加法），确认日志中出现技能调用且结果正确。
4. **因果链**：写入前因、经过、结果三条事件，提问「为什么会有这个结果」，预期回答会按时间顺序串联。
5. **修正与刷新**：修改一条记忆后再次检索，确认答案随之更新，证明缓存与存储同步。

## 常见查询示例
- **事实类**：`agentdb recall --query "谁提交了理赔表单"`
- **总结类**：`agentdb recall --query "本周广州的关键事件"`
- **因果类**：`agentdb causal --query "为什么广州订单被加急"`
- **技能类**：`agentdb skill math-cskill --args "(120-30)/3"`

## 排查指引
- 若回答不引用最新记忆，检查检索阈值或向量库是否同步。
- 若技能未触发，确认技能配置文件中的触发器与命名是否一致，并查看日志。
- 若因果回答过于模糊，增加时间戳或明确事件角色，再次写入后重试。
