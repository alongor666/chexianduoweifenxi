# 用户收益指南

## 文档目的

本指南旨在用简洁的语言说明 Insuralytics 为不同角色带来的具体收益，帮助新用户快速理解「为什么要用」以及「能带来什么价值」。

## 适用角色

- **精算师 / 数据分析师**：需要快速验证计算逻辑、评估成本结构，并生成报告。
- **业务 / 渠道负责人**：关注签单进度、渠道贡献和结构优化，要求直观的趋势与结构视图。
- **管理层**：希望通过少量关键指标即可把握业务健康度，便于决策与资源调配。

## 关键收益

1. **一目了然的业务健康度**：KPI 仪表盘以 4x4 网格展示综合成本率、签单进度、边际贡献等核心指标，帮助管理层在分钟级完成业务体检。
2. **趋势洞察与对比**：提供核心 KPI 的历史曲线以及多周导入/对比能力，便于识别周期波动、同比/环比异常与策略效果。
3. **多维健康度雷达图**：从边贡率、进度达成、赔付率、出险率、费用率五个维度综合评分，可快速定位拉低整体表现的具体维度。
4. **灵活的筛选与下钻**：按机构、险种、渠道等维度进行切片分析，同时支持结构分析（如车型、渠道占比），让问题定位从「发现异常」到「锁定原因」更高效。
5. **可靠的数据流与校验**：
   - 支持多周同时导入与智能去重，保证数据连贯且避免重复劳动。
   - 客户端与服务端双重校验、导入结果统计与历史追溯，提升数据可信度。
   - 独立的计算校验工具确保前端展示与后端计算一致，降低决策风险。
6. **报告与共享友好**：分析结果可导出 CSV，便于与外部系统或线下报告结合；仪表盘与图表布局清晰，适合在汇报场景直接展示。

## 典型使用场景

- **周度经营复盘**：导入最新周次数据，查看 KPI 仪表盘与趋势图，快速完成周会汇报材料的核心数据部分。
- **渠道策略评估**：通过筛选渠道与险种，结合结构分析和雷达图，评估政策调整或促销活动的效果。
- **风险与成本控制**：跟踪赔付率、出险率、费用率的异常波动，及时采取定价或费用控制措施。
- **数据对账与核验**：利用计算校验工具与导入历史记录，对前后端指标、不同周次或不同来源的数据进行一致性检查。

## 快速上手提示

1. **准备数据**：按项目提供的 CSV 模板整理周次数据，确保必填字段完整且格式正确。
2. **导入与验证**：使用「多周导入」上传数据，查看导入结果统计与错误提示，必要时修正并重新导入。
3. **设定筛选器**：在仪表盘/分析页面设置机构、险种、渠道等筛选维度，以匹配当前关注的业务范围。
4. **查看关键视图**：
   - KPI 仪表盘：先看综合成本率与边际贡献，快速判断整体健康度。
   - 趋势分析与雷达图：用于发现波动、定位维度问题。
   - 结构分析与数据导出：在需要深挖或生成报告时使用。

## 最佳实践

- **小步验证**：每次导入或筛选调整后，先检查关键 KPI 和导入统计，确认数据可信再深入分析。
- **与团队共享视图**：固定关键筛选器（如核心渠道/地区），形成对齐的周度或月度视图，减少沟通成本。
- **更新开发文档**：当数据字段或计算逻辑有调整时，及时同步到 `开发文档/` 相关章节，确保团队认知一致。

## 常见问题与建议

- 若导入文件包含历史周次，系统会自动跳过已导入数据并给出统计，避免重复计算。
- 计算结果如与外部系统有差异，可使用校验工具逐项核对输入与公式，或查看导入历史以排查数据缺失。
- 欢迎将使用体验与需求反馈记录到项目 issue，便于持续迭代用户体验与分析深度。
