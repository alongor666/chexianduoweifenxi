# 项目概览

## 项目名称
车险多维可视化智能数据分析网页（Insurance Analytics Platform）

## 项目目的
一个基于 Web 的车险数据分析平台，用于：
- 上传和解析车险业务数据（CSV/JSON格式）
- 可视化展示核心 KPI 指标
- 多维度筛选和趋势分析
- 结构化对比分析
- 数据导出和报告生成

## 核心功能模块
1. **数据导入模块 (F001)**: CSV/JSON 文件上传与解析
2. **KPI 看板模块 (F002)**: 8个核心业务指标展示
3. **趋势分析模块 (F003)**: 52周时间序列分析
4. **多维筛选模块 (F004)**: 11个业务维度筛选
5. **结构分析模块 (F005)**: 对比与分群分析
6. **数据导出模块 (F006)**: CSV/PNG/PDF 导出
7. **计算验证模块 (F007)**: 公式透明化展示

## 当前状态
- 版本: MVP v1.0 (Beta)
- 部署环境: 开发环境
- 平均功能完整度: 73%

## 技术特点
- 全客户端处理，支持离线使用
- 大数据量处理（支持100万+行数据）
- 使用 DuckDB-WASM 进行高性能 SQL 查询
- 响应式设计，支持多种设备
